# Projeto Aplicado I - Análise da Superlotação no Sistema Metroviário de São Paulo

## Objetivo
Analisar a demanda média de passageiros por estação no metrô de São Paulo, com base em dados públicos de 2025 e 2026, para identificar padrões de concentração de passageiros e possíveis indícios de superlotação.

## Base de dados
- Demanda de Passageiros por Estação – Média Dias Úteis – 2025
- Demanda de Passageiros por Estação – Média Dias Úteis – 2026

## Tecnologias utilizadas
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn ou Plotly, se tiver usado

## Etapas do projeto
1. Coleta dos dados
2. Limpeza e tratamento
3. Transformação para formato analítico
4. Análise exploratória
5. Geração de tabelas e gráficos
6. Interpretação dos resultados

## Estrutura do repositório
- `/notebooks` → scripts em Jupyter Notebook
- `/resultados` → tabelas e gráficos exportados
- `README.md` → descrição do projeto
